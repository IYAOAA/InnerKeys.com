WELCOME TO INNERKEYS.COM ECOMMERCE SETUP

This folder contains a simple, mobile-friendly ecommerce website starter.

Files:
- index.html — homepage
- styles.css — styling
- products.html — showcase items
- cart.html — cart page
- about.html — about us
- contact.html — contact form

TO USE:
1. Unzip this folder
2. Open index.html in your browser
3. Add product images manually in the folder if needed
4. You can customize by editing the .html files in a code editor like VS Code
5. Upload to Netlify, GitHub Pages, or Hostinger to go live

Pages can be extended with JS and databases in the future for full functionality.
